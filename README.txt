WARNING:
Running the following commands is going to replace the existing figures
---------------------------------------------------------------------------

Reproducing the results / figures:
To reproduce the results you can run the scripts q1 and q2 respectively.

Locating the figures:
The figures are stored in the Q1 and Q2 directories

For Q1:
go to ./Q1/Connectivity ton find the Connectivity matrices
go to ./Q1/Firing_MFR ton find the raster plots of the firings and the plots for the Mean firing rate

For Q2:
go to ./Q2/neural_complexity.fig for the neural complexity graph (and the SWI)

If you want to run the Q2 trials again you can run SimulateTrials(n,t)
for n trials and time t
